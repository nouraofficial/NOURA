About These Numbers
AI usage is currently tracked per-device (this browser only) since there's no backend yet to aggregate requests across every user. Platform-wide AI monitoring — popular prompts, failure rates, per-user cost — needs requests to route through a backend proxy (planned alongside the Gemini key migration). Once that exists, this page will show real cross-platform numbers instead of this device's local count.

Role-Based Access Control
These are the roles Noura's admin system is designed around. Actual enforcement (limiting which admin can do what) requires backend authentication with per-account roles — this panel is single-login for now, so every admin currently has full access. Treat this as the spec to build against once accounts move to a real backend.
